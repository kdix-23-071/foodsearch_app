## 簡易仕様書

### アプリ名:MogMap(もぐまっぷ)

### 対象 OS: Android 16

### 開発環境: AndroidStudio

### 使用言語: Kotlin Java

### 開発期間: 2 週間(11/7~11/21)

### 機能一覧

現在地からの範囲を指定して店舗を検索する
検索結果を一覧表示する
検索結果に表示されたカードをタップすると、その店舗の詳細情報を表示する

### 画面概要

#### 検索画面

- 検索範囲を 5 段階のうちから指定する
- 位置情報を取得する
- 位置情報・選択範囲をもとに店舗を検索する

#### 検索結果画面

- 現在地・選択範囲をもとに取得した情報をカードにまとめて表示する
- 表示している情報は、店舗写真・店舗名・アクセス情報

#### 店舗詳細画面

- 検索結果画面で表示されたカードをタップすると、より詳細な店舗の情報を表示する
- 表示している情報は、店舗写真・住所・営業日・アクセス・ジャンル・予算・座席数・ホームページ・駐車場情報・携帯アクセス
  コンセプト: カフェ・スイーツに特化した検索アプリ

### 使用しているライブラリ・SDK など

- Android SDK
- JetpackCompose
  UI やボタンなどのマテリアルを使用

- Retrofit
  API 通信クライアント (com.squareup.retrofit2)
  ホットペッパーグルメ API との通信に使用
- OkHttp
  HTTP クライアント。ログ出力用の logging-interceptor も使用
- Gson
  JSON パースライブラリ (com.google.code.gson)
  API レスポンスを Kotlin オブジェクトに変換するために使用
- Coil
  画像ロードライブラリ (io.coil-kt)
  API から取得した画像を表示するために使用
- Hilt
  依存関係注入ライブラリ (com.google.dagger:hilt-android)
  ViewModel や Repository の管理に使用
- Google Play Services Location
  位置情報取得ライブラリ (com.google.android.gms:play-services-location)
  現在地を取得するために使用
- Accompanist Permissions
  Jetpack Compose でのパーミッション処理を簡単にするライブラリ (com.google.accompanist)

### こだわったポイント

検索する前に必ず位置情報を取得するので、食べ歩きやカフェ巡りで頻繁に移動するときにおすすめです

「位置情報がずれてて、遠くのお店を検索してしまった」という事故を防ぎます
